Layer 4

Core Modules (4 files, ~600 lines total)

SceneManager.ts

Three.js Scene, PerspectiveCamera (45Â° FOV), WebGLRenderer (antialias, shadows, SRGB). Warm lighting rig: AmbientLight(0xffeedd, 0.6) + DirectionalLight(0xffffff, 1.0). Camera at (0, 4, 8) looking at (0, 1, 0). Per-task camera presets in scene configs. Resize handler. Render loop.

AssetLoader.ts

GLTFLoader with Map cache. preloadTask(taskId) loads all characters, props, environments from manifest. Clone-on-use pattern: cache originals, clone() for scene instances. Progress callback for loading bar UI.

AnimationController.ts

AnimationMixer per character. Loads 161 clips from shared animation GLB. play(name, fadeIn=0.25) with crossFadeTo transitions. update(delta) called in render loop. Fallback to Idle if clip not found (never crashes).

SceneScriptPlayer.ts

Takes SceneScript JSON, executes actions sequentially with delay_ms timing. spawn â†’ load + position model. animate â†’ play clip. move â†’ tween position. react â†’ trigger Lottie. sound â†’ play audio. camera â†’ tween camera. Emits "script-complete" event when done.
