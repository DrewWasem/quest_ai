# Glossary — Domain Vocabulary

## Terms We Use (Kid-Facing)

| Term | Meaning | Notes |
|------|---------|-------|
| **Quest** | A game scenario/task | Never "challenge", "exercise", "level" |
| **Description** | What the kid types | NEVER "prompt" — the word "prompt" does not exist in kid-facing text |
| **Plan** | Alternative to "description" | "Describe your plan" is more natural for some tasks |
| **Go!** | Submit button label | Never "Submit", "Send", "Enter" |
| **Try it!** | Alternative submit label | Implies experimentation, not completion |
| **Try something else** | Retry button | Never "Try again" (implies they failed) |

## Terms We Use (Internal/Dev)

| Term | Meaning |
|------|---------|
| **Scene script** | The JSON that Claude generates to describe a 3D animation |
| **Narration** | The short text displayed after the scene plays (max 20 words) |
| **Feedback tip** | The concrete suggestion displayed after narration |
| **Vocabulary contract** | The list of actors, props, animations, and effects Claude can reference |
| **Golden cache** | Pre-computed Claude responses for common inputs |
| **Funny fail** | A FUNNY_FAIL success_level — vague input → hilarious chaos |
| **Full success** | A FULL_SUCCESS — kid covered all key elements |

## Terms We NEVER Use (Kid-Facing)

| Banned Term | Why | Say Instead |
|-------------|-----|-------------|
| Prompt | Technical jargon kids don't know | "description" or "plan" |
| AI / Artificial Intelligence | Breaks the magic; kids don't need to know | (don't reference it at all) |
| Skills | Sounds like a classroom | (don't reference it at all) |
| Learning | Makes it feel like homework | (don't reference it at all) |
| Challenge | Implies difficulty/pressure | "quest" |
| Wrong | Implies failure/shame | (describe what happened instead) |
| Error | Technical/scary | (describe what happened instead) |
| Failed | Devastating to a kid | (describe the funny outcome instead) |
| Mistake | Implies blame | (describe the funny outcome instead) |
| Level | Implies hierarchy/gatekeeping | "quest" |
| Score | Implies competition/judgment | (don't score) |
| Submit | Corporate/formal | "Go!" or "Try it!" |
| Great job! | Empty praise — kids see through it | (describe the specific cool thing that happened) |
